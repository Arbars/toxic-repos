# toxic-repos

В последнее время участились случаи добавления в популярные Open Source проекты бекдоров разной степени опасности и полит агиток нацеленных на жителей РФ и россиян по всему миру.
Это может нести угрозу людям, а также подрывает доверие к Open Source комьюнити и Open Source как таковому.
